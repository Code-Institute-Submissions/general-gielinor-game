What is the
General Gielinor Game?

General Gielinor Game is a JavaScript quiz designed to teach you about the MMORPG RuneScape, you'll learn about some parts of the game that might help you out. It was developed by me, Joeyy Scott.

RuneScape was founded in 2001 and coded in the programming language Java. It's an MMORPG that allows you to start a new character and forge your own path within the world of Gielinor.

What is the
General Gielinor Game?

General Gielinor Game is a JavaScript quiz designed to teach you about the MMORPG RuneScape and developed by me, Joeyy Scott.

RuneScape was founded in 2001 and coded in the programming language Java, not Javascript as you'll see here! It's an MMORPG (Massively multiplayer online role playing game) that allows you to start a new character and forge your own path within the world of Gielinor. Throughout the questions in this quiz you'll learn about some parts of the game that might help you out and teach you about aspects of the game you didn't know about!
